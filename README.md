PAC-MAN on Shoes
================

This is a tiny Shoes app written under 99 steps for the 30th anniversary of PAC-MAN!

**Note**: Need Policeman included Ruby 1.9

Usage
-----

	cd pacman_on_shoes
	shoes pacman_on_shoes.rb
